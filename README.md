## 💻 Sobre o Projeto

O objetivo é demonstrar o uso de dois prompts, que poderiam ser transformados em agentes, para identificar a carreira profissional na carreira de tecnologia mais adequada e gerar um plano de estudos. Foi usado o chatGPT utilizando os modos adequados para cada prompt e objetivo (pense bem, pesquisa profunda, pesquisar na web), primeiro identificando a carreira mais adequada, depois realizando um plano de estudos de 90 dias incluindo links para cursos gratuitos sobre os tópicos selecionado e uma planilha Excel com esse plano de estudo. O plano de estudo foi feito para uma pessoa fictícia que tem interesse na área de dados e está começando do zero.

## 🎯 Resultados

### 🎙Etapa 1: entrevista e identificar a carreira

Foi usado o prompt do AGENT 1 com sete perguntas, usando o modo pensamento profundo. A nossa pessoa fictícia:

- Tem mais interesse em entender sistemas = - está começando do zero.
- Consegue dedicar 24 horas por semana aos estudos.
- Prefere lidar com dados no dia a dia.
- Tem como objetivo conseguir o primeiro emprego.
- Tem mais interesse em dados, inteligência artificial e banco de dados.
- Não possui experiência profissional prévia.

O resultado deste primeiro agente foi:

Com base no seu perfil, identifiquei 3 carreiras muito promissoras:

════════════════════════════════════════════════════════════
🥇 1º LUGAR: Analista de Dados - 18/20
════════════════════════════════════════════════════════════

💡 POR QUE COMBINA COM VOCÊ:
Você demonstrou interesse em entender sistemas, trabalhar com dados e explorar inteligência artificial. Além disso, sua disponibilidade de 24 horas semanais ajuda bastante no aprendizado de ferramentas essenciais da área. Como está começando do zero, análise de dados oferece uma entrada relativamente acessível no mercado tech.

⚖️ O QUE ESPERAR:

VANTAGENS:

Forte conexão com dados e tomada de decisão
Boa porta de entrada para áreas de IA no futuro
Desenvolvimento de raciocínio analítico e visão de negócios

DESAFIOS:

Exige prática constante com dados reais
Necessidade de aprender ferramentas e visualização
Pode envolver bastante interpretação de regras de negócio

📈 MERCADO:
A demanda por profissionais de dados continua crescendo em muitas empresas, embora as oportunidades variem bastante conforme região, experiência e setor.

════════════════════════════════════════════════════════════
🥈 2º LUGAR: Administrador de Banco de Dados (DBA) / Engenharia de Dados - 17/20
════════════════════════════════════════════════════════════

💡 POR QUE COMBINA COM VOCÊ:
Seu interesse em entender sistemas e banco de dados indica afinidade com estruturas, organização de informação e funcionamento interno das aplicações. Essa área costuma agradar pessoas mais analíticas e detalhistas.

⚖️ O QUE ESPERAR:

VANTAGENS:

Forte contato com sistemas e arquitetura
Conhecimento muito valorizado em empresas
Base sólida para crescer em cloud e engenharia de dados

DESAFIOS:

Curva de aprendizado técnica mais intensa
Exige atenção a performance e segurança
Algumas funções podem exigir responsabilidade operacional alta

📈 MERCADO:
Profissionais especializados em dados estruturados e infraestrutura de dados seguem importantes em empresas de diversos tamanhos, especialmente em ambientes com grande volume de informação.

════════════════════════════════════════════════════════════
🥉 3º LUGAR: Engenheiro de Machine Learning / IA - 15/20
════════════════════════════════════════════════════════════

💡 POR QUE COMBINA COM VOCÊ:
Você demonstrou interesse direto em inteligência artificial e dados. Essa é uma área extremamente interessante para quem gosta de entender sistemas complexos e comportamento de informações.

⚖️ O QUE ESPERAR:

VANTAGENS:

Área inovadora e em expansão
Contato com tecnologias modernas e automação inteligente
Possibilidade de atuar em projetos muito variados

DESAFIOS:

Exige base forte em programação e matemática
Caminho até a primeira vaga costuma ser mais longo
Necessidade constante de atualização

📈 MERCADO:
IA continua recebendo investimentos e atenção do mercado, mas as vagas de entrada podem ser mais competitivas e geralmente exigem uma base técnica mais robusta.

Também foi feito o prompt: 

- Com base nas minhas respostas poderia montar uma pesquisa de mercado com vagas para estagiários e júniores em início de carreira?

Gerando uma resposta completa e longa com os mercados que fazem mais sentido para o usuário (Analista de Dados Jr., estágio em dados/BI, Engenharia de dados Jr., mais a médio prazo) assim como o panorama atual do mercado no Brasil e o que as empresas geralmente buscam nos candidatos iniciantes, as hard e soft skills mais requisitadas, exemplos de vagas recentes com os requisitos listados e cidades onde aparecem mais vagas para iniciantes.

Destaque para o que as vagas para iniciantes mais pedem:

** Hard skills mais frequentes **

- SQL
- Python
- Excel avançado
- Power BI
- Git/GitHub
- Estatística básica
- Banco de dados relacionais

** Soft skills mais valorizadas **

- Raciocínio analítico;
- Organização;
- Comunicação de insights;
- Autonomia para aprender.

### 📖 Etapa 2: gerar plano de estudos

Foi usada a resposta obtida pelo AGENT 1 como entrada para o AGENTE 2, em um contexto novo. 
O chatGPT pegou automaticamente as respostas da etapa 1.

O prompt fez um plano para um analista de dados, mostrando como é o trabalho de um analista de dados, mapa de habilidades essenciais e desejáveis, ferramentas para dominar, roadmap de 90 dias, projeto de portfólio(tratar e analisar uma base pública de vendas, criando insights e uma dashboard), roteiro de entrevista para um cargo iniciante, trilha do DIO recomendada (Coding The Future Squadio - Python Data Analytics) e recursos online recomendados (Kaggle Learn - Python, Kaggle Learn - Intro to SQL, Microsoft Learn - Training for Data Analysts, Microsoft Learn - Training for Power BI, freeCodeCamp - Data Analysis with Python, PostgreSQL Tutorial).

O resumo da trilha sugerida, gerado por IA, está no arquivo [trilha_resumo](roadmap_resumo.xlsx).
Um roadmap mais completo, gerado usando um esforço maior e listando recursos, está no arquivo [roadmap_completo](roadmap_completo.xlsx).
